04 — Network Strategy
Why Point of Service (POS)?
Out-of-State Employees
0
Zero Impact
POS acts exactly like a PPO for any employee outside the network state. No referrals required. No changes to how they access care.
In-State Employees
Minimal Admin Difference
Only difference: employees select a primary care physician (PCP). Same doctors, same hospitals, same network. Most already have one.
PPO
See any in-network provider. No PCP required. Self-refer to specialists. Higher premiums.
POS
Same network. Choose a PCP. Can still self-refer in most plans. Lower premiums.
Bottom Line: Same doctors, same hospitals, same coverage — with meaningful premium savings.
